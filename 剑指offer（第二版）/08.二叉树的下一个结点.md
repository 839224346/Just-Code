### 8.二叉树的下一个结点
#### 题目描述
给定一个二叉树和其中的一个结点，请找出中序遍历顺序的下一个结点并且返回。注意，树中的结点不仅包含左右子结点，同时包含指向父结点的指针。
#### 解法：

```python
# class TreeLinkNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None
#         self.next = None
class Solution:
    def GetNext(self, pNode):  # 这道题情况一定要考虑全
        if not pNode:
            return None
        if pNode.right:        # 如果有右结点，那么此时pNode是根结点，
                               # 下一个结点是右子树的最左边的结点
            pNode = pNode.right
            while pNode.left:
                pNode = pNode.left
            return pNode
        while pNode.next:      # 用来判断当前结点是不是根结点
            if pNode.next.left == pNode:
                               # 如果是左结点，那么返回它的父结点
                return pNode.next
            pNode = pNode.next # 如果不是，再向上找
        return pNode.next
```
