### 22.链表中倒数第 k 个结点
#### 题目描述
输入一个链表，输出该链表中倒数第k个结点。
#### 解法：

```python
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def FindKthToTail(self, head, k):
        if not head or k <= 0:   # 判断条件head为空或者k<=0都不是常规满足的条件
            return None
        pre = post = head        # 快慢指针做这道题
        while k:                 # 快指针先走 k 步
            if pre:              # 加判断是防止 k 的数值比链表的长度长
                pre = pre.next
                k -= 1
            else:
                return None
        while pre:               # 快慢指针一起走，保持 k 个距离
            pre = pre.next
            post = post.next
        return post
```
